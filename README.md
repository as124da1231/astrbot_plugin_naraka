# astrbot_plugin_naraka

永劫无间战绩查询插件，数据源为第三方战绩站 `naraka.drivod.top`（小程序数据源）。仅适配 QQ（aiocqhttp）。

## 指令

```
战绩查询 <昵称或UID> [模式]
详细查询 <昵称或UID>
永劫战绩 <昵称或UID> [模式]    水墨风战绩图片（别名：战绩图）
```

- `<昵称或UID>`：游戏昵称（走搜索）或纯数字 UID（roleIdSimple）。
- `[模式]`（战绩查询 / 永劫战绩）：`天选/匹配/天人` + `单排/双排/三排`，可连写或简写。不填则按最近一场自动判断当前模式。
- 回复会 @ 发起查询的人。
- 昵称与 UID 同时显示：无论用昵称还是 UID 查询，结果都会显示「昵称」与「UID」两行（拿不到昵称时昵称显示“未知”）。

### 战绩查询（概览）

包含：排位（天选）/天人各三档分数、当前模式段位与分数、最近 N 场场均伤害/振刀/击败、常用英雄、当前模式的队友统计。

- 自己的数据（常用英雄、场均）：基于最近返回的全部对局（约 10 场，混排）。
- 队友统计：只统计与当前模式（大类+人数都一致）相同的对局；单排不显示、双排 1 个、三排 2 个；该模式无对局则空。

### 永劫战绩（图片）

与「战绩查询」内容相同，渲染为水墨风格图片。常用英雄带圆形英雄头像（在线加载，加载失败时以英雄名首字兜底）。

- 渲染依赖框架内置的 HTML 渲染能力（`self.html_render`）。若渲染失败或环境不支持，会自动降级为文字版概览，不会无响应。
- 背景为内置水墨画（`img/naraka_bg.png`），所有玩家共用同一张。

### 详细查询

最近 N 场逐场明细（名次、英雄、振刀、伤害、时间）+ 底部场均与常用英雄。

## 模式代码（gameMode / subtype）

| | 单排 | 双排 | 三排 |
|---|---|---|---|
| 天选 | 101 | 102 | 103 |
| 匹配 | 201 | 202 | 203 |
| 天人 | 401 | 402 | 403 |

## 配置项

| 配置 | 说明 | 默认 |
|---|---|---|
| `image_quality` | 战绩图片清晰度：`normal` 普通 / `high` 高清 / `ultra` 超清 | high |
| `timeout` | 单接口请求超时（秒） | 15 |
| `cache_ttl` | 结果缓存时长（秒） | 60 |
| `default_season_id` | 赛季 ID 兜底值 | 9620020 |

## 说明与限制

- 振刀、队友数据需对每场对局调用单场详情接口；一次查询会发起较多请求（已用并发 + 单场详情长期缓存优化）。请勿高频使用，以免对站点造成压力或被限流。
- 单场详情任一场查询失败时，为保证数据准确会提示“查询失败”，而非展示不完整数据。
- 受站点数据源限制，最近对局通常仅最近约 10 场，且包含各模式（单/双/三排）。
